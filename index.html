搬送経路カール予測モデル

用紙の粘弾性を2並列Maxwellモデルで表現し、 搬送中の温度・含水率変化と曲げ曲率の時間履歴から最終カール量を予測します。

// 2並列Maxwellモデル構成則
σ_h + (μ_h/E_h)·dσ_h/dt = μ_h·dε/dt ...(1)
σ_s + (μ_s/E_s)·dσ_s/dt = μ_s·dε/dt ...(2)
σ = σ_h + σ_s ...(3)

// 温度・含水率依存性
μ_h(T,W) = μ_h0·exp(-T/T_μh)·exp(-W/W_μh) ...(4)

// 伸縮率カール係数
Ce = dγ/dW ...(5)

// 曲率更新（離散化）
σ'_h^(n+1) = A_h·σ'_h^n + B_h·Δγ ...(9)
σ'_s^(n+1) = A_s·σ'_s^n + B_s·Δγ ...(10)
Δγ = Δγ_p - Δγ_c ...(12)
📌 検証結果：普通紙0.94、再生紙0.90、厚紙0.96の相関係数でカール量を予測可能
デカーラ矯正量予測モデル

用紙を粘弾塑性モデル（バネ＋スライダー＋ダッシュポット）で表現。 塑性変形と粘性変形の重ね合わせによりカール矯正メカニズムを再現します。

// 粘弾塑性構成則（各要素 i）
ε̇_i = σ̇_i/E_i (|σ_i| ≤ σ_Yi) ...(19)
ε̇_i = σ̇_i/E_i + (σ_i-σ_Yi)/μ_i ...(20)
(|σ_i| > σ_Yi)
σ = Σ σ_i ...(21)

// カール矯正量への寄与（支配因子）
① 変形曲率（ローラー曲率に依存）
② 変形時間（搬送速度に反比例）
③ 圧力・引張力は影響小
📌 検証結果：普通紙・厚紙ともに相関係数0.90以上。パラメーター設計工数を約80%削減達成
カール発生メカニズム分類
Ⅰ 表裏伸縮率差カール
脱湿時の収縮度合いが表裏で異なることによって生じる
Ⅱ 定着後カール
定着器通過時の熱と変形ストレスによって発生
Ⅲ 搬送経路カール
搬送中に曲げられた状態で脱湿硬化し形状を記憶
Ⅳ 大変形カール（デカーラ矯正）
大きな曲率での強制曲げによる永久変形でカールを矯正
測定手法

TMA（熱機械分析）
温湿度制御下での応力-歪み関係測定。高湿条件でのDMA破断問題を回避。

伸縮率カール係数測定
重力垂直方向配置の試験片で、脱湿量とカール曲率の線形関係を測定。

定着後カール分離法
用紙を表裏反転して2回測定し平均を取ることで、表裏伸縮率差カールを相殺。

有限要素法（Abaqus）
デカーラ前後の位置-曲率関係を構造解析で算出。
